# FastAPI Microservices Project

Данный проект содержит два микросервиса, разработанных с использованием FastAPI:
- **TODO-сервис** для управления задачами.
- **Сервис сокращения URL** для создания коротких ссылок.

Оба микросервиса используют SQLite для хранения данных и контейнеризованы с помощью Docker.

---

## Требования

- Python 3.9+
- Docker и Docker Compose
- SQLite

---

## Запуск через Docker

Запуск с использованием Docker
1. Соберите контейнеры
```
docker-compose build
```

2. Запустите контейнеры
```
docker-compose up -d
```

3. Проверка работоспособности
   
* TODO-сервис: http://localhost:8000/docs
* Сервис сокращения URL: http://localhost:8001/docs
